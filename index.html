Reported Speech — Control System

Commands: told + object + to + verb
Statements: said (that) + clause
Backshift: can → could

1. Build it (Commands)

“Listen carefully.”
“Open the window.”
“Sit down.”
“Write the answer.”

2. Build it (Statements + Backshift)

“You can change the world.”
“We can win this.”
“They can do it.”

3. Combine

“Stand on your desks.”
“Words can change the world.”
